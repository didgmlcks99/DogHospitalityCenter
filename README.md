# DogHospitalityCenter
## 프로그램의 주제
#### 유기견 보호 센터에서 유기견 데이터 및 정보 정리
+ 활용 중인 데이터
    * 견종
    * 나이
	* 들어온 날짜
	* 성향
	* 예방 접종 여부
	* 발견 (시)
	* 중성화 유무

## 대략 기능
+ 기본 기능
    * create (추가)
	* read (출력)
	* update (수정)
	* delete (삭제)
	* save (저장) 
	* load (불러오기)
	* list (여러 데이터 출력)
+ 추가 기능
    * 견종별 검색
	* 나이 검색
	* 들어온 날짜
	* 발경 시
